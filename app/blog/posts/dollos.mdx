## Dependencies

The following Python packages are required to run DollOS:

- `random`
- `binrobotdollos`
- `morsildollos`
- `dollpycode`
- `dollnotepad`
- `rich`
- `subprocess`
- `psutil`
- `json`
- `shutil`
- `datetime`
- `time`

## Setup

1. Clone the repository from GitHub:

    ```bash
    git clone https://github.com/dollengo/dollos.git
    cd dollos
    ```

2. Install the necessary dependencies:

    ```bash
    pip install rich psutil
    ```

3. Run the `start()` function to begin:

    ```bash
    python dollos.py
    ```

## Commands

DollOS supports a variety of commands. Use the `.help` command to display the list of available commands. Some key commands include:

- `.desktop`: Show the Desktop
- `.help`: Show help for using the operating system
- `.powershell`: Execute Windows PowerShell commands
- `.exit`: Exit the system
- `.list`: List files in the current directory
- `.read [filename]`: Read a file
- `.write [filename]`: Write to a file
- `.rename [old_name] [new_name]`: Rename a file
- `.mkdir [dirname]`: Create a directory
- `.rmdir [dirname]`: Remove a directory
- `.delete [filename]`: Delete a file
- `.move [src] [dest]`: Move a file
- `.copy [src] [dest]`: Copy a file
- `.cd [dirname]`: Change directory
- `.datetime`: Display date and time
- `.history`: Show command history
- `.usage`: Show CPU and memory usage
- `.netstat`: Show network statistics
- `.ps`: Show running processes
- `.kill [pid]`: Kill a process by its PID
- `.touch [filename]`: Create an empty file
- `.cat [filename]`: Display file content
- `.head [filename] [n]`: Display the first `n` lines of a file
- `.tail [filename] [n]`: Display the last `n` lines of a file
- `.du`: Display disk usage
- `.uptime`: Show system uptime
- `.env`: Show environment variables
- `.setenv [key] [value]`: Set an environment variable
- `.unsetenv [key]`: Unset an environment variable
- `.path`: Show the system PATH
- `.addpath [path]`: Add a directory to the system PATH
- `.removepath [path]`: Remove a directory from the system PATH
- `.ports`: List open ports
- `.ipconfig`: Show IP configuration
- `.ping [host]`: Ping a host
- `.traceroute [host]`: Trace the route to a host
- `.hostname`: Show the hostname
- `.arp`: Show the ARP table
- `.route`: Show the routing table
- `.systeminfo`: Show system information
- `.diskpart`: Manage disk partitions
- `.mount [device] [path]`: Mount a device
- `.umount [path]`: Unmount a device
- `.chmod [mode] [filename]`: Change file permissions
- `.chown [user:group] [filename]`: Change file owner
- `.lsusb`: List USB devices
- `.lspci`: List PCI devices
- `.dmesg`: Display system messages
- `.date`: Display the date
- `.time`: Display the time
- `.man [command]`: Show the manual for a command
- `.alias [name] [command]`: Create an alias
- `.unalias [name]`: Remove an alias
- `.su [user]`: Switch user
- `.sudo [command]`: Execute a command as another user
- `.install [package]`: Install a package
- `.uninstall [package]`: Uninstall a package
- `.update`: Update packages
- `.upgrade`: Upgrade packages
- `.services`: List services
- `.start [service]`: Start a service
- `.stop [service]`: Stop a service
- `.restart`: Restart the system
- `.enable [service]`: Enable a service
- `.disable [service]`: Disable a service
- `.status [service]`: Display service status
- `.log [service]`: Display service log
- `.journal`: Show the system journal
- `.save [filename]`: Save the current state
- `.load [filename]`: Load a saved state
- `.shutdown`: Shutdown the system
- `.reboot`: Reboot the system
- `.ldi`: List device information
- `.errors`: List error codes
- `.dolley`: Create a new text file with encrypted data
- `.opendolley`: Read the dolley file
- `.savestate`: Save the current state (aliases, desktoparray, etc.)
- `.loadstate`: Load the saved state
